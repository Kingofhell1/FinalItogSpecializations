### Для создания файлов с помощью команды cat в терминале операционной системы Linux необходимо выполнить следующие команды:

* Создание файла "Домашние животные" и заполнение его содержимым:
  * cat > Pet
* Создание файла "Вьючные животные" и заполнение его содержимым:
  * cat > Pack
* Объединение файлов с помощью команды cat:
  * cat Pet Pack > Animals
* Просмотр содержимого созданного файла:
   * cat Animals
* Переименование файла "Друзья человека" на "Друзья животных":
   * mv Animals Human_Friends
### Для создания директории и перемещения файла в нее в терминале операционной системы Linux можно выполнить следующие команды:

* Создание директории "my_directory":
  * sudo mkdir Human_Friends1
* Перемещение файла "Друзья животных" в созданную директорию:
  * sudo mv Human_Friends Human_Friends1



### Для подключения дополнительного репозитория MySQL можно выполнить следующие шаги:


* Добавим ключ репозитория с помощью команды:
  * sudo wget -O- https://repo.mysql.com/RPM-GPG-KEY-mysql-2022 | sudo apt-key add -
* Добавим репозиторий MySQL в систему с помощью команды:
   * sudo echo "deb http://repo.mysql.com/apt/ubuntu/ $(lsb_release -sc) mysql-8.0" | sudo tee /etc/apt/sources.list.d/mysql.list
* Обновим список пакетов с помощью команды:
   * sudo apt update
* Установим любой пакет из репозитория MySQL, например, пакет mysql-shell, можно с помощью команды:
   * sudo apt install mysql-shell
### Для установки и удаления deb-пакета с помощью dpkg можно выполнить следующие шаги:

* Скачаем deb-пакет, например, Midnight Commander (MC), с помощью команды:
   * sudo wget http://archive.ubuntu.com/ubuntu/pool/universe/m/mc/mc_4.8.25-2ubuntu2_amd64.deb
* Установим deb-пакет с помощью команды:
   * sudo dpkg -i mc_4.8.25-2ubuntu2_amd64.deb
* Удалим установленный пакет с помощью команды:
   * sudo dpkg -r mc
